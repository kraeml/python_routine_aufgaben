
# Vergleichsoperatoren

| Operator | Bedeutung |
|----------|-----------|
| == | Gleich |
| != | Ungleich |
| < | Kleiner als |
| > | Größer als |
| <= | Kleiner oder gleich |
| >= | Größer oder gleich |
